## 🛒 Zepto Product Analytics Dashboard

## 📌 Project Overview

This project analyzes Zepto's product catalog to uncover insights related to pricing, discounts, inventory availability, and category performance using SQL and Power BI.

The objective of this project is to transform raw product data into actionable business insights that can support inventory management, pricing strategies, and decision-making.

## 🎯 Business Problem

Retail businesses need to monitor product pricing, discount strategies, and inventory levels to maximize profitability and improve customer satisfaction.

This project answers key business questions such as:

- Which categories contain the highest number of products?
- Which categories offer the highest discounts?
- Which products are the most expensive?
- Which categories have the highest number of out-of-stock products?
- Which categories contribute the highest inventory value?
- Which products provide the largest savings to customers?

The dataset contains product-level information from Zepto, including:

| Column Name | Description |

| Available Quantity | Stock Available |
| Discounted Selling Price | Final Selling Price |
| Weight In Gms | Product Weight |
| Out Of Stock | Stock Availability Status |

## 🔍 SQL Analysis Performed

### 1. Premium Product Analysis
Identified products with the highest MRP to understand the premium product segment.

### 2. Discount Analysis
Analyzed products receiving the highest discounts.

### 3. Category-wise Average Discount Analysis
Compared average discount percentages across categories.

### 4. Out-of-Stock Analysis
Identified categories with the highest number of unavailable products.

### 5. Category Distribution Analysis
Analyzed product count across categories.

### 6. Inventory Level Analysis
Measured total available inventory by category.

### 7. Inventory Value Analysis
Calculated category-wise inventory value using available stock and selling prices.

### 8. Customer Savings Analysis
Identified products offering the highest monetary savings.

### 9. Category-wise Pricing Analysis
Compared average MRP across categories.

### 10. Full Price Product Analysis
Measured the number of products sold without any discount.

## 📈 Key Insights

- Analyzed over 3,000+ products across 14 categories.
- Fruits & Vegetables category offered the highest average discounts.
- Cooking Essentials and Munchies contained the largest number of products.
- More than 450 products were identified as out of stock.
- Premium products reached prices above ₹2500.
- Significant inventory value was concentrated in a few major categories.
